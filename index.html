Lessons Learned - trybe

Sobre Lições

Em pouco mais de duas semanas na trybe, nos já desbravamos diversos assuntos. Para o desenvolvedor, é fundamental dominar a linha de comando, por isso, aprendemos a usar Unix e Bash. Nesse bloco, estudamos comandos para criar pastas, criar arquivos, navegar entre pastas, alterar arquivos e olhar o status desses arquivos usando a linha de comando do linux. Após o bloco de Unix e Bash, passamos pelo bloco de git e github. O git é maravilhoso e é uma ferramenta utilizada para criar versões do código através de comandos de git como o branch, para criar ramificações do código e push e pull para puxar e empurrar as versões para o repositório remoto além de colaborar com repositórios de outros usuários. Além disso, aprendemos os fundamentos da internet para compreender melhor os temas que abrangem o desenvolvimento web. Dentro do bloco, estudamos métodos do protocolo HTTP, requisição, resposta, relação entre cliente e servidor, DNS e outros elementos de internet Na segunda semana, começamos os fundamentos de HTML e CSS para estruturar e adicionar estilo ao site. HTML é uma linguagem de marcação de texto que é utilizada para estruturar páginas na web. Estudamos sobre as tags da linguagem e como usar essas tags para estruturar o código. Estudamos também as tags semânticas para descrever o significado de conteúdos especificos como cabeçalhos, rodapes, artigos, seções e artigo. E o CSS para posicionar os elementos dentro da página, mudar as cores, fontes e outros elementos de estilo e adicionar estilo às tags com as classes, seletores e ids. Paralelamente, aprendemos sobre soft skills e como lidar com relações inter-pesssoais dentro dos círculos corporativos e de trabalho. Aprendemos a desenvolver competências como empatia, pro-atividade e inteligência emocional entre outras habilidades.
