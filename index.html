Darciano Utzig, graduado em Design Industrial e Administração, resolveu trocar de ramo e decicar-se a programação, estudando pra ser desenvolvedor Full-Stack na Trybe. Como parte dos conhecimentos adquiridos, inicialmente no curso citado, desenvolveu este site como projeto de final de módulo. O presentte site usa os conhecimentos de HTML e CSS, bem como a utilizacão do GitHub para versionamento dos códigos desenvolvidos.

Este documento marca o início do desenvolvimento web. Foi com a elaboração desta singela página que iniciou a caminhada.
