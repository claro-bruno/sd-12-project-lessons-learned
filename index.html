    Minha experiência na Trybe

    Até agora na Trybe, tenho aprendido muitas ferramentas importantes para uma pessoa programadora. Dentre elas, como já mencionado na lista de lições aprendidas, estão: a utilização do Git e do GitHub para versionar códigos, o HTML para estruturar páginas e o CSS para estilizá-las. Além disso, tenho aprendido a desenvolver resiliência, frente aos inúmeros desafios encontrados pela pessoa desenvolvedora ao longo de sua jornada.
